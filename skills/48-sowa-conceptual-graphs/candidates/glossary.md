# Candidate Glossary

- Abstract syntax: notation-independent representation of semantically significant CG features.
- Actor: conceptual relation used to represent a computable or externally determined function.
- Arc: ordered link between a relation node and a concept node.
- CGIF: Conceptual Graph Interchange Format, the machine interchange syntax.
- CLIF: Common Logic Interchange Format.
- Common Logic: ISO/IEC 24707 semantic foundation for CGIF, CLIF, and XCL.
- Concept: node that refers to an entity, set, or range, with type and referent.
- Conceptual graph: bipartite graph of concepts and conceptual relations.
- Conceptual relation: relation node with valence and signature.
- Context: concept containing a nested graph used for scope, quotation, negation, conditionals, propositions, and situations.
- Coreference label: defining `*x` or bound `?x` label used to mark identity.
- Defined quantifier: quantifier such as `@every` that expands into core graph structures.
- Descriptor: nested CG used to describe a referent.
- Designator: literal, locator, name, individual marker, or indexical identifying a referent.
- Display form: graphical human notation.
- Linear form: compact human-readable linear notation.
- Relation hierarchy: partial order of relation labels with valence and signatures.
- Signature: ordered type constraints for relation arguments or lambda parameters.
- Type hierarchy: partial order of concept type labels.
